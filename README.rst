A Discord bot that collects attendance declarations for upcoming workdays.

Features:

- Per-channel RSVP tracking
- Button-based attendance selection
- Deadline-based reminders
- Admin-managed directory of participants

📖 Documentation: https://gterreran.github.io/Workday_RSVP_Discord_Bot/

----

Installation
------------

To install the latest version directly from GitHub:

.. code-block:: bash

   pip install git+https://github.com/gterreran/Workday_RSVP_Discord_Bot.git

It is recommended to first create and activate a clean environment using Python ≥3.10.
See the full `Documentation <https://gterreran.github.io/https://gterreran.github.io/Workday_RSVP_Discord_Bot/getting-started/installation.html>`_ for more details.
